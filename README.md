## 🌌 Overview

**Gemini Been Space** is a single-page web application that serves as a visually appealing, highly legible guide for Indonesian students. It provides clear, step-by-step instructions on how to activate a 1-year free trial of Google Gemini Advanced using `.ac.id` or `.edu` email addresses. 

Built entirely with HTML and styled via the powerful **Tailwind CSS** framework, the interface utilizes a glassmorphism design approach with smooth, pulsing gradient backgrounds and a sleek `Inter` font typography.

## 🔮 Features

- 💅 **Glassmorphism UI** — Semi-transparent, blur-backed components that look stunning against animated gradients.
- 🎨 **Tailwind CSS** — Fully responsive, utility-first styling requiring no external CSS files.
- ⚡ **Lightweight** — A single-file architecture (`panduangemini.html`) optimized for instant loading.
- 📱 **Mobile-Ready** — Perfect formatting on desktops, tablets, and mobile devices.
- 🎓 **Educational Utility** — Clear, actionable insights for students aiming to maximize their academic workflow with AI.

## 🛠️ Stack & Technologies

* **HTML5** — Semantic structuring.
* **Tailwind CSS** (via CDN) — Rapid UI design.
* **Google Fonts (Inter)** — Crisp, modern typography.

## 🚀 Quick Start

Since this project requires zero build steps, deploying or viewing it locally is effortless.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/geminibeenspace.git
   ```
2. **Open the file:**
   Simply double-click `panduangemini.html` or open it in your preferred browser.
3. **Customize:**
   Modify the custom Tailwind config directly within the `<script>` tag in the `<head>` section to tweak colors like the signature `#ff66c4` (pink-accent).

## 🌑 Aesthetic Guidelines

This project embraces a modern, ethereal aesthetic. The background incorporates subtle pink and purple blur pulses (`animate-pulse`) to represent the ever-evolving, intelligent nature of Gemini AI, creating an engaging visual experience that keeps the reader focused.
